# Crimes Exemplares: An Interactive Exploration

This project is an interactive digital experience inspired by Max Aub's book "Crimes Exemplares". It was created by Carolina Silva, Joana Oliveira, and João Matos in 2024.

## Overview

Explore a virtual living room where each object reveals a piece of a mysterious narrative. By clicking on objects, users uncover clues, images, and audio fragments, gradually reconstructing the story behind the crimes.

## Features

- Immersive interactive environment with clickable objects
- Visual and audio clues for each item
- Ambient sounds and background music
- Responsive design for various screen sizes
- Animated transitions and popups

## How to Use

1. Open `p3livingroom.html` in your web browser.
2. Click the **Explorar** button to start.
3. Interact with objects in the room by clicking on them to reveal more information.
4. Use the mute button to toggle sound, and the back button to return to the home screen.
5. Find all objects to complete the experience.

## Project Structure

- `p3livingroom.html`: Main HTML file for the interactive experience
- `cc/`: Images for revealed objects and alternate room states
- `cf/`: Popup images for each clue
- `sc/`: Scene and object images for the initial state
- `sfx/`: Sound effects and background music
- `som/`: Audio clues for each object
